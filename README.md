# 📝To-Do App

A minimal and responsive To-Do List app built with pure vanilla JavaScript — no frameworks, no dependencies.

## ✨ Features
- ➕ Add new tasks instantly
- ✅ Mark tasks as complete with a single click
- 🗑️ Delete tasks you no longer need
- 💾 Tasks saved automatically via localStorage — persist even after page refresh
- 🌑 Sleek dark mode UI out of the box

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 How to Run
1. Clone the repo
2. Open `index.html` in your browser
3. That's it — no installs needed!

## 🧠 What I Learned
- DOM manipulation and event handling
- localStorage for client-side data persistence
- Managing UI state without any library or framework
